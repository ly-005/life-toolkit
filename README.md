# 🧰 生活效率工具箱

一个功能完整的生活效率管理工具，包含待办清单、记账本、习惯打卡等实用功能。

## 🌐 在线访问

[https://ly-005.github.io/life-toolkit/](https://ly-005.github.io/life-toolkit/)

## ✨ 功能特性

### 📝 待办清单
- 添加待办事项，支持高/中/低优先级
- 标记完成/未完成
- 筛选显示（全部/未完成/已完成）
- 实时统计总数、完成数和完成率

### 💰 记账本
- 记录收入和支出
- 实时显示收入、支出和结余
- 按时间倒序展示记录
- 支持删除记录

### 🎯 习惯打卡
- 添加习惯并设定目标天数
- 每日打卡
- 可视化进度条和最近7天打卡日历
- 自动统计打卡进度

### 📊 使用统计
- 工具使用次数统计
- 待办数据概览
- 财务统计概览
- 习惯进度统计

### 👤 关于我
- 个人简历展示页面

## 🎨 其他功能

- 🌙 深色/浅色主题切换
- 💾 本地存储，刷新不丢失数据
- 📤 数据导出（JSON格式）
- 📥 数据导入
- ⌨️ 键盘快捷键（Ctrl+1~5切换工具）
- 🔗 快捷操作按钮
- 📱 响应式设计，完美适配手机端

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`
2. 或者访问 GitHub Pages 在线链接

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+1 | 待办清单 |
| Ctrl+2 | 记账本 |
| Ctrl+3 | 习惯打卡 |
| Ctrl+4 | 使用统计 |
| Ctrl+5 | 关于我 |
